
metadata.pull\_from\_objkt
==========================

This module can be used to download Tezos NFT metadata from objkt.com.
Here is how to download Tezzardz metadata from the terminal:

.. code-block:: shell

   $ python3 metadata/pull_from_objkt.py -contract KT1LHHLso8zQWQWg1HUukajdxxbkGfNoHjh6
